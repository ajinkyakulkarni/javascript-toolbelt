  JST

System Time

system_time returns highest resolution time offered by host in milliseconds. 


 JST.system_time()

Elapsed Time

time returns a string with the number of msec resolution time offered by host in milliseconds.

JST.time(function(){ Math.sqrt(1232); })

Memoization

memoize returns a memoized version of a 1-arity function.

heavy_calculation = function(x){ return Math.sqrt(Math.sin(x) + Math.cos(x)) }; memo = JST.memoize(heavy_calculation); [ JST.time(function() { heavy_calculation(1232); }), JST.time(function() { memo(1232); }), JST.time(function() { memo(1232); })]
As you can see, the first call to memo takes about the same time as the direct call to heavy_calculation. But subsequent calls are very fast.
